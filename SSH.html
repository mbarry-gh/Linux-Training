SSH, the Secure Shell : The SSH system, is a protocol for remote logins and for securing network services on an insecure network. SSH’s capabilities include remote command execution,shell access, file transfer, port forwarding, network proxy services,and even VPN tunneling. SSH is a client/server protocol that uses cryptography for authentication, confidentiality, and integrity of communications between two hosts. OpenSSH essentials : • ssh, the client • sshd, the server daemon • ssh-keygen, for generating public/private key pairs • ssh-add and ssh-agent, tools for managing authentication keys • ssh-keyscan, for retrieving public keys from servers • sftp-server, the server process for file transfer over SFTP • sftp and scp, file transfer client utilities
